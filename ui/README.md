# QML Components

This folder contains the QML components for UI extensibility and reusability.

# Components

## MainWindow
AnyNoise is a single-window desktop app, MainWindow is that "single window".


## MediaToolBar
The MediaToolBar is the bottom toolbar for controlling sound playing, volume and visualizing metadata.
